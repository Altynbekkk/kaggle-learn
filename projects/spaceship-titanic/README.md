# Spaceship Titanic

**Kaggle Competition:** [Spaceship Titanic](https://www.kaggle.com/competitions/spaceship-titanic)

**Description:** Binary classification competition where we predict whether passengers are transported by the alien spaceship or not.

**Approach:**
- Data preprocessing and feature engineering
- Handling missing values (HomePlanet, Destination, Age)
- XGBoost classifier for prediction
- Feature encoding and selection

**Files:**
- `spaceship-titanic.ipynb` - Complete solution notebook
