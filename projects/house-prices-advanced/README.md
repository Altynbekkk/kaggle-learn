# House Prices - Advanced Regression Techniques

**Kaggle Competition:** [House Prices - Advanced Regression Techniques](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques)

**Description:** Predict sales prices of houses using advanced regression techniques with challenging dataset containing many features and data quality issues.

**Approach:**
- Feature selection from 80+ variables
- One-hot encoding for categorical features
- XGBoost Regressor with hyperparameter tuning
- Handling missing values and feature reindexing

**Files:**
- `house-prices-advanced.ipynb` - Complete solution with model training and predictions
